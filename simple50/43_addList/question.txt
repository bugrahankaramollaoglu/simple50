Add to Linked List

Write a function that adds a new node at the end of a linked list.

Example:
Input: 1 -> 2 -> 3 -> null, value = 4
Output: 1 -> 2 -> 3 -> 4 -> null

Input: null, value = 5
Output: 5 -> null

Input: 1 -> null, value = 2
Output: 1 -> 2 -> null

Note: Add new node at the end of the list.
