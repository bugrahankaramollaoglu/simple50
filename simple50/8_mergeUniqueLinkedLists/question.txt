Merge Unique Linked Lists

Write a function that merges two linked lists and removes duplicates.

Example:
Input: list1 = [1, 2, 3, 4], list2 = [3, 4, 5, 6]
Output: [1, 2, 3, 4, 5, 6]

Input: list1 = [1, 1, 2, 2], list2 = [2, 3, 3, 4]
Output: [1, 2, 3, 4]

Note: Return a new list with unique elements from both lists.
