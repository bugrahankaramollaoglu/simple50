Check if Arrays Equal

Write a function that checks if two arrays are equal.

Example:
Input: [1, 2, 3], [1, 2, 3]
Output: true

Input: [1, 2, 3], [3, 2, 1]
Output: false

Input: [1, 2], [1, 2, 3]
Output: false

Input: [], []
Output: true

Input: [1], [1]
Output: true

Note: Arrays must have same length and elements in same order.
