Check if Array is Sorted

Write a function that checks if an array is sorted in ascending order.

Example:
Input: [1, 2, 3, 4, 5]
Output: true

Input: [5, 4, 3, 2, 1]
Output: false

Input: [1, 1, 2, 2, 3]
Output: true

Input: [1]
Output: true

Input: []
Output: true

Note: Consider duplicate elements as sorted.
