Rotate Array

Write a function that rotates an array to the right by k positions.

Example:
Input: arr = [1, 2, 3, 4, 5], k = 2
Output: [4, 5, 1, 2, 3]

Input: arr = [1, 2, 3, 4], k = 1
Output: [4, 1, 2, 3]

Input: arr = [1, 2, 3], k = 3
Output: [1, 2, 3]

Note: k can be larger than array length. Handle this case.
