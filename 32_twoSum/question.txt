Two Sum

Write a function that finds two numbers in an array that add up to a target sum.

Example:
Input: [2, 7, 11, 15], target = 9
Output: [0, 1]

Input: [3, 2, 4], target = 6
Output: [1, 2]

Input: [3, 3], target = 6
Output: [0, 1]

Input: [1, 5, 8, 10, 13], target = 18
Output: [2, 4]

Input: [1, 2, 3, 4, 5], target = 9
Output: [3, 4]

Note: Return indices of the two numbers. Assume exactly one solution exists.
